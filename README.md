#### 🔧 아두이노 기반 커스텀 디지털 계기판 개발
* **프로젝트 개요:**
    * 자작 자동차의 핵심 데이터를 드라이버에게 실시간으로 제공하기 위해,<br>
      C/C++(Arduino)를 활용하여 디지털 계기판을 직접 설계하고 구현했습니다.
* **사용 기술:** `C/C++`, `Arduino`, GPS 모듈, LCD 디스플레이
* **프로젝트 목표:**
    1. GPS, RPM 등 차량의 핵심 센서 데이터 수집
    2. 수집된 데이터를 가공하여 LCD 계기판에 실시간으로 표시

* **핵심 구현 내용:**
    * **(배선 작업)** 차량 전체의 전원 분배 및 센서 데이터 통신을 위한 배선 작업을 주도했습니다.
    * **(데이터 수신)** 아두이노를 허브로 사용하여 **GPS 센서**로부터 위도, 경도, 시간 데이터를 `Serial` 통신으로 수신했습니다.
    * **(데이터 가공)** 수신된 GPS 데이터와 별도의 RPM 신호를 입력받아, 드라이버에게 필요한 **속도, 시간, RPM** 정보로 가공했습니다.
    * **(문제 해결: 필터링)** GPS 수신 초기에 **값이 비정상적으로 튀는** 현상을 발견했습니다. 이를 해결하기 위해, 특정 임계값을 벗어나는 데이터는 무시하고 이전 정상 값을 유지시키는 간단한 **필터링 로직**을 C코드로 직접 구현했습니다.
    * **(시각화)** 가공된 최종 데이터를 LCD 디스플레이에 포맷팅하여 실시간으로 출력했습니다.

* **배운 점:**
    * 컴퓨터공학 전공 지식이 실제 하드웨어와 어떻게 연결되는지 실전으로 경험했습니다.
    * C언어를 이용한 데이터 처리, 하드웨어 제어, 시리얼 통신에 대한 깊은 이해를 얻었습니다.
    * 데이터 시트를 분석하고, 튀는 값과 같은 실제 문제를 정의하고 해결하는 디버깅 역량을 길렀습니다.
